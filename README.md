# Restaurant-Review-Sentiment-Analyzer-
 A Streamlit web app that predicts sentiment (Positive, Negative, Neutral) from restaurant feedback using an SVM model with TF-IDF. Features live text analysis, animated backgrounds, and AI-driven insights for customer experience improvement.
